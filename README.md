# Vibe Color Extension

A simple browser extension that changes the background color of the current webpage to a random color when you click the extension icon or the popup button.

## How to use
1. Load the extension in your browser (Chrome/Edge):
 - Go to Extensions > Manage Extensions > Load unpacked
 - Select this folder
2. Click the extension icon or "Vibe!" button to change the page color.
